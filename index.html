🤖 Aviator Telegram Funnel Bot (Kenya)

Bot hii ya Telegram inawaongoza watumiaji hatua kwa hatua ili kujisajili kwenye 1xBet, kuweka amana, na kuanza kupata pesa kupitia mchezo wa Aviator 🛩️

✨ Features:

  • Lugha ya Kiswahili 🇰🇪
  • Promo code support (KN611)
  • Screenshot submission & user guide
  • Referral system (with your own link)

⚙️ Setup:

1. Clone repo
2. Create .env file with your Telegram token:
TELEGRAM_TOKEN=your_token_here
3. Install dependencies:
pip install -r requirements.txt
4. Run bot:
python bot.py
